What's New in Astro Fetch v0.0.4

  • Optimized Performance: Improved game performance to run smoothly on more devices.

  • Background Music: Introduced different background music for enhanced gameplay experience.

  • Refined Movement System: Made movement smoother and easier to control.

  • Increased Speed and Rotation: Enhanced player speed and rotation for a more dynamic gameplay.

  • New Levels: Added 15 new levels, including an updated version of the original level.

  • Story and Intro: Created an introductory sequence and started integrating a story into the game.

  • New Animations and Texts: Added many new animations and texts in the game including animations and texts appearing after clearing levels.

  • Project Structure Improvements: Streamlined project structure for faster development and easier updates.

  • Unused Packages Removed: Optimized game by removing unused packages.

  • New Background Color: Replaced the old background color with a darker one.

  • HUD Enhancements: Added stats at the top of the screen (current level and remaining boxes) and a version number with a message at the bottom.

  • Credits Screen: Added a credits screen (still a work in progress).

  • Sound Effects: Implemented sound effects for collisions and pickups (still a work in progress, not shipped with this update).

  • Main Menu: Developed and shipped a main menu screen.

      • Future Features:

    • Tutorials: Work on tutorials is in progress and will be included in the next update.

    • Level Progression Save: A feature to save user progress is being developed and is hoped to be available in update v0.0.5 or v0.0.6.
